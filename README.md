## node 加速镜像配置

下载 `.npmrc`

```bash
wget -O ~/.npmrc https://cdn.jsdelivr.net/gh/wencaizhang/.config/.npmrc
```

> 注意：这会覆盖你用户目录下的 `.npmrc` 文件，因此为了便于恢复到你自己的配置，请做好备份。下同。

下载 `.yarnrc`

```bash
wget -O ~/.yarnrc https://cdn.jsdelivr.net/gh/wencaizhang/.config/.yarnrc
```
